What This Does

  • Converts 1.21 vanilla resource packs to Tuff Client format
  • Removes unnecessary files (models, blockstates, sounds)
  • Maintains texture compatibility with Tuff Client
  • Preserves all block, item, and entity textures
  • Updates pack metadata to match Tuff standards

Important Notes

  • Client-side only: All processing happens in your browser
  • No upload: Your files never leave your computer
  • Texture style: This converter maintains original textures
  • Manual review: Always test the output in-game

Style Conversion

This tool converts the structure of resource packs, not the art style. This converter focuses on file organization and Tuff Client compatibility.
