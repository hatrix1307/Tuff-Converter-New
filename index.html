How This Works

  • Upload the 1.12 version of your texture pack for reference textures
  • Upload the 1.21 version of the same pack for new content
  • Converter merges both: 1.21 textures + 1.12 animated/legacy textures
  • Creates legacy name duplicates for full Tuff Client compatibility
  • Outputs a complete Tuff Client-ready resource pack

Important Notes

  • Client-side only: All processing happens in your browser
  • No upload: Your files never leave your computer
  • Both packs needed: Use the same texture pack, just different versions
  • Manual review: Always test the output in-game

What Gets Merged

The 1.12 pack provides: animated water/lava, legacy entity textures, and properly formatted GUI elements. The 1.21 pack provides: all new blocks, items, and textures from updates 1.13+. Both are combined to create the perfect Tuff Client pack.
